# Archive

This directory holds frozen historical material for the legacy He-LMAS track.

## Contents

- `legacy-he-lmas/`: the original heterogeneous KV-bridge project entry point and notes.
- `legacy-he-lmas/code/`: frozen legacy root code moved from the repository top level (`configs/`, `scripts/`, `src/`, `tests/`).

The archive is read-only in practice: new work should happen under the active `helmas3n/` code and the `paper/` draft.
