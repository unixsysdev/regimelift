# Archive

This directory holds frozen historical material and completed experiment snapshots.

## Contents

- `legacy-he-lmas/`: the original heterogeneous KV-bridge project entry point and notes.
- `experiments/targeted_site_study_v1/`: the first completed HeLMAS-3n targeted site study, including the decision memo that established `layer34,last1` as the first robust learned handoff site.

The archive is read-only in practice: new work should happen under the active `helmas3n/` code and the `paper/` draft.
